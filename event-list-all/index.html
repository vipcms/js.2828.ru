Расширенная таблица событий JavaScript

Автор: Профессор JS · Дата: 05 апреля 2025 г.

Все современные и мобильные события, включая WebSocket, IndexedDB и Drag & Drop API.

События

Категория Событие Описание Пример использования Примечание
Мобильные wheel Прокрутка колесиком мыши или тачпадом window.addEventListener('wheel', handler) Поддерживает touchpad на мобильных устройствах
orientationchange Изменение ориентации экрана window.addEventListener('orientationchange', handler) Доступно только на мобильных устройствах
WebSockets open Соединение установлено socket.addEventListener('open', handler) Вызывается один раз при успешном подключении
message Получено сообщение от сервера socket.addEventListener('message', handler) Часто используется для обработки данных
error Ошибка соединения socket.addEventListener('error', handler) Нужно корректно обработать
close Соединение закрыто socket.addEventListener('close', handler) Вызывается при любом закрытии
reconnect Пользовательское событие повторного подключения — Не стандартное, но часто реализуется вручную
IndexedDB success Успешное открытие базы данных request.addEventListener('success', handler) Обязательное событие при работе с IDB
error Ошибка при операциях с базой данных request.addEventListener('error', handler) Требует обработки
upgradeneeded Требуется обновление версии базы indexedDB.open(...).addEventListener('upgradeneeded', handler) Вызывается при изменении версии БД
blocked Открытие блокировано из-за старых соединений request.addEventListener('blocked', handler) Важно для управления версиями
complete Завершение операции записи transaction.addEventListener('complete', handler) Используется после добавления/обновления записей
abort Операция прервана transaction.addEventListener('abort', handler) Может быть вызвано пользователем или ошибкой
Drag and Drop dragstart Начало перетаскивания el.addEventListener('dragstart', handler) Важно установить данные через dataTransfer
drag Процесс перетаскивания el.addEventListener('drag', handler) Вызывается постоянно во время перетаскивания
dragend Окончание перетаскивания el.addEventListener('dragend', handler) Вызывается как на источнике, так и на цели
dragenter Курсор входит в зону дропа target.addEventListener('dragenter', handler) Часто используется вместе с dragover
dragover Курсор над зоной дропа target.addEventListener('dragover', handler) Нужно предотвратить дефолтное поведение
drop Объект "упал" на целевой элемент target.addEventListener('drop', handler) Основное событие для получения данных
dragleave Курсор покидает зону дропа target.addEventListener('dragleave', handler) Можно использовать для стилизации
