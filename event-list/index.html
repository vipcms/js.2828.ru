Список всех основных событий в JavaScript (HTML5)

Автор: Профессор JS · Дата: 05 апреля 2025 г.

Введение

События в JavaScript — это способ отслеживать действия пользователя и изменения в браузере. В этой статье приведена полная таблица часто используемых событий, разбитых по категориям.

Таблица событий

Категория Событие Описание Пример использования
События мыши click Клик левой кнопкой мыши element.addEventListener('click', handler)
dblclick Двойной клик element.addEventListener('dblclick', handler)
mousedown Нажатие кнопки мыши element.addEventListener('mousedown', handler)
mouseup Отпускание кнопки мыши element.addEventListener('mouseup', handler)
mousemove Перемещение курсора над элементом element.addEventListener('mousemove', handler)
mouseover Курсор наведён на элемент element.addEventListener('mouseover', handler)
mouseout Курсор уходит с элемента element.addEventListener('mouseout', handler)
contextmenu Правый клик / контекстное меню element.addEventListener('contextmenu', handler)
Pointer события pointerdown Начало взаимодействия указателем el.addEventListener('pointerdown', handler)
pointerup Завершение взаимодействия указателем el.addEventListener('pointerup', handler)
pointermove Перемещение указателя el.addEventListener('pointermove', handler)
pointercancel Взаимодействие прервано системой el.addEventListener('pointercancel', handler)
pointerover Указатель наведён на элемент el.addEventListener('pointerover', handler)
pointerout Указатель ушёл с элемента el.addEventListener('pointerout', handler)
Touch события touchstart Начало касания экрана el.addEventListener('touchstart', handler)
touchend Конец касания el.addEventListener('touchend', handler)
touchmove Перемещение пальца по экрану el.addEventListener('touchmove', handler)
touchcancel Касание было прервано el.addEventListener('touchcancel', handler)
Формы и элементы ввода input Изменение значения поля ввода (в реальном времени) input.addEventListener('input', handler)
change Изменение значения после потери фокуса input.addEventListener('change', handler)
focus Элемент получил фокус input.addEventListener('focus', handler)
blur Элемент потерял фокус input.addEventListener('blur', handler)
submit Отправка формы form.addEventListener('submit', handler)
reset Форма была сброшена form.addEventListener('reset', handler)
Клавиатура keydown Нажата клавиша (повторяется при удержании) window.addEventListener('keydown', handler)
keyup Отпущена клавиша window.addEventListener('keyup', handler)
keypress Введён символ (устаревшее) —
Окно и загрузка страницы load Страница полностью загружена window.addEventListener('load', handler)
DOMContentLoaded DOM готов к работе document.addEventListener('DOMContentLoaded', handler)
beforeunload Пользователь покидает страницу window.addEventListener('beforeunload', handler)
unload Страница выгружается window.addEventListener('unload', handler)
resize Изменение размеров окна window.addEventListener('resize', handler)
scroll Прокрутка элемента или окна window.addEventListener('scroll', handler)
Мультимедиа play Воспроизведение запущено video.addEventListener('play', handler)
pause Воспроизведение остановлено video.addEventListener('pause', handler)
ended Медиа закончилось audio.addEventListener('ended', handler)
volumechange Изменение громкости video.addEventListener('volumechange', handler)
timeupdate Изменение текущего времени воспроизведения video.addEventListener('timeupdate', handler)
